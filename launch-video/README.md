# Ecosystem Intelligence Launch Video

90-second Remotion launch video for Ecosystem Intelligence.

## Commands

```bash
npm run dev
npm run still
npm run render
```

## Composition

- `LaunchVideo`
- 1920 x 1080
- 30 fps
- 2700 frames

The video uses captured app screenshots from `public/screenshots/` and follows the launch storyboard: problem, product reveal, discovery modes, engagement targeting, evidence trace, boundaries, outcome, and CTA.

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm run dev
```

**Render video**

```console
npx remotion render
```

**Upgrade Remotion**

```console
npx remotion upgrade
```
